WHITESPACE FIRST THEME
http://my.studiopress.com/themes/whitespace/

INSTALL
1. Upload the Whitespace theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Whitespace Pro theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

CUSTOM BACKGROUND
This theme supports custom background, which is a native WordPress function.
You can find some really great free textures to use at http://subtlepatterns.com.
